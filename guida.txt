INSTALLAZIONE

scaricare i pacchetti necessari dal file 'requirements.txt'
scaricare il modello 'SO_word2vec', che è mantenuto in un file .bin della dimensione di 1.5 GB, al seguente link: http://doi.org/10.5281/zenodo.1199620
collocare il modello dentro la directory 'project'

ESECUZIONE

eseguire il file 'gui.py'

selezionare il file di requisiti 'finto_data_set.txt' tramite il bottone Browse
selezionare l'algoritmo desiderato tramite il bottone Configure
una volta selezionato l'algoritmo, si aprirà una finstra che ci permetterà di configurare i parametri
premere il bottone Ok senza fare nient'altro per eseguire l'algoritmo con i parametri settati al loro valore di default
premere Run
attendere la visualizzazione della finestra contenente gli outputs
è possibile selezionare le frasi (tutte o alcune) per poi salvarle tramite il bottone Save o per rieseguire un algoritmo su di esse tramite il bottone Rerun
in quest'ultimo caso tuttavia è necessario prima riconfigurare l'algoritmo tramite il bottone Reconfigure
